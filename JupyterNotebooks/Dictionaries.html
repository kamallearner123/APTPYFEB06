üîπ What is a Dictionary?

A dictionary in Python is an unordered collection of key-value pairs. It allows for fast lookups and modifications.

üîπ Creating a Dictionary

my_dict = {'name': 'Alice', 'age': 25, 'city': 'New York'}

üîπ Accessing Values

print(my_dict['name'])  # Output: Alice

üîπ Adding or Modifying Values

my_dict['age'] = 26
my_dict['job'] = 'Engineer'

üîπ Removing Elements

del my_dict['city']

üîπ Dictionary Methods

  • dict.keys() - Returns all keys.
  • dict.values() - Returns all values.
  • dict.items() - Returns key-value pairs.
  • dict.get(key, default) - Fetches a key, or default if not found.
  • dict.pop(key) - Removes a key and returns its value.
  • dict.popitem() - Removes and returns the last key-value pair.
  • dict.update(new_dict) - Merges dictionaries.
  • dict.clear() - Removes all items.

üîπ Example Usage

Looping through keys & values

for key, value in my_dict.items():
    print(key, ":", value)

Merging Dictionaries

dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}
dict1.update(dict2)
print(dict1)  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}
